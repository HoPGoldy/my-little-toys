# my-little-toys

个人兴趣制作的一些小玩意，使用 `vue` + `element` + `leaflet` + `animjs` 制作，你可以通过 [点击这里](https://hopgoldy.github.io/my-little-toys/) 来浏览它们！

# 依赖

- node.js `@8.10.0` 或更高
- npm `@5.6.0` 或更高
- yarn `@latest`
- vue-cli `@3.1.3` 或更高


# 使用

**安装依赖**

```
yarn install
```

**运行**

```
yarn serve
```

**构建**

```
npm run build
```
